# News App

## Overview
News App is a Streamlit-based web application that allows users to browse and summarize the latest news articles on various topics. It leverages the Google Generative AI model to provide concise summaries of news descriptions, enhancing the reading experience.

## Features
- **Topic Selection**: Users can select from predefined news categories such as World, Nation, Business, Technology, Entertainment, Sports, Science, and Health.
- **Search Functionality**: Users can search for news articles on any topic of their interest.
- **News Summarization**: The app provides a summary of each news article using a generative AI model.
- **Responsive UI**: The app has a user-friendly interface that works well on both desktop and mobile devices.
